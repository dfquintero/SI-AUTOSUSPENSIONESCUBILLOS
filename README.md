# Autosuspensiones Cubillos 🚗🔧
> **Versión 1.0.0**

Sistema web integral diseñado para la gestión operativa, control de inventarios, administración de clientes y automatización de órdenes de trabajo para el taller mecánico **Autosuspensiones Cubillos**.

## 🛠️ Tecnologías Utilizadas
El sistema está construido bajo una arquitectura robusta y escalable utilizando:

*   **Backend:** Java, Maven (Gestión de dependencias)
*   **Frontend:** HTML5, CSS3, JavaScript (Interfaces dinámicas y responsivas)
*   **Servidor/Despliegue:** Entorno local compatible con servidores de aplicaciones Java (como Apache Tomcat)

---

## 🚀 Características Principales
*   **Gestión de Inventario:** Control en tiempo real de repuestos y herramientas.
*   **Administración de Clientes:** Historial de vehículos y datos de contacto.
*   **Órdenes de Trabajo:** Creación, seguimiento y actualización del estado de los servicios.
*   **Facturación:** Generación de recibos y cálculo de costos de mano de obra y repuestos.

## 💻 Instalación y Configuración Local

### Requisitos Previos
*   Java JDK 8 o superior.
*   Apache Maven instalado.
*   Servidor local (Ej: Apache Tomcat).

### Pasos para Ejecutar
1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com
    ```
2.  **Compilar el proyecto con Maven:**
    Navega a la raíz del proyecto y ejecuta:
    ```bash
    mvn clean install
    ```
3.  **Despliegue:**
    Despliega el archivo `.war` generado en tu servidor local de preferencia y accede a través del navegador en `http://localhost:8080/SI-AUTOSUSPENSIONESCUBILLOS` (o el puerto configurado).
